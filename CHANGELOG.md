# Changelog

All notable changes to this project are documented here. The project follows
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-07-22

### Added

- Mandatory question-first intake for every lifecycle mode.
- Explicit context-and-assumptions confirmation gate before work.
- Cognitive-load review at the start and end of each lifecycle step.
- Complete source map covering the book preface and all 23 chapters.
- Cross-harness installer for Codex, Claude Code, and Antigravity (`agy`).
- Repository validation, reproducible release packaging, and CI workflows.

### Changed

- Tightened outputs to one active decision, three material risks, and one next
  gate to reduce extraneous and intrinsic load.
- Separated facts, assumptions, unknowns, decisions, and actions throughout
  the templates.

## [1.1.0] - 2026-07-21

### Added

- Three end-to-end modes: Plan, Manage/intervene, and Interpret/analyze.
- Experiment control board and stage-specific plan and analysis templates.
- Dependency-free sample-size, mean-effect, and SRM calculator.

## [1.0.0] - 2026-07-20

### Added

- Initial trustworthy online controlled experiment workflow grounded in
  Kohavi, Tang, and Xu.
